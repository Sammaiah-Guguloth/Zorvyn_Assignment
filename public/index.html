About the Assignment

This project is a back-end RESTful API submission for the Zorvyn full-stack assignment. It is designed to manage financial records with a focus on security, role-based access control (RBAC), and analytics data processing.

Technologies Used

Node.js Express.js MongoDB Mongoose JWT Zod BcryptJS Jest

Available APIs

Authentication

Secure login and password management using bcrypt and JWT.

Admin Management

Manage user accounts, roles, and status through restricted endpoints.

Financial Records

Complete CRUD operations for transactions with soft-delete support.

Analytics

Aggregated data summaries and monthly financial trends via MongoDB pipelines.

How to Start

  • Run npm run dev to start the server locally.
  • Visit /api-docs to see the interactive Swagger UI.
  • Use the Postman links above for a collection and detailed docs.
  • The default API prefix is /api/v1.
